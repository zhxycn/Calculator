# Calculator

A simple calculator built with Qt

## Features

- Arbitrary-precision arithmetic (GNU MP) for accurate numerical results
- Standard operators (+ − × ÷) with parentheses and correct operator precedence
- Keyboard input with mouse-friendly buttons
- Clean, responsive, and accessible Qt-based GUI for desktop
- Cross-platform desktop support: Windows, macOS, Linux

## Build

### Prerequisites

- Qt 6
- GNU MP (GMP) bignum library

### Steps

1. Open the project in Qt Creator.
2. Configure the build settings as needed.
3. Click on the "Run" button to build and run the application.
